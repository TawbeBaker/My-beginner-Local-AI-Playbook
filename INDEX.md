## 📱 Quick Reference

### Commands Cheat Sheet

```bash
# Ollama
ollama pull llama3.2        # Download model
ollama run llama3.2         # Run model
ollama list                 # List installed models

# Docker
docker-compose up -d        # Start services
docker-compose down         # Stop services
docker ps                   # List running containers

# Python
pip install -r requirements.txt  # Install dependencies
python chatbot.py           # Run chatbot

# Whisper
whisper audio.mp3 --model medium  # Transcribe audio
```

### Port Reference

```
n8n:          5678
Qdrant:       6333
PostgreSQL:   5432
Ollama API:   11434
```

### File Locations

```
Ollama models:    ~/.ollama/models/
SD models:        (Stability Matrix folder)/models/
Docker volumes:   Docker Desktop → Volumes
Python packages:  (venv)/lib/python*/site-packages/
```
